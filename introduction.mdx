# Welcome to Product Builders Landing

Product Builders Landing is a modern, responsive Next.js 15 landing page designed for showcasing product consultancy and mentorship services. Built with TypeScript, Tailwind CSS, and shadcn/ui components, it provides a professional platform to attract and convert potential clients.

## What's included

The landing page includes several key sections:

- **Hero section** - Eye-catching introduction with headline and call-to-action
- **Services section** - Three-column grid showcasing your core services
- **Offerings section** - Detailed cards for individual programs and mentorships
- **Contact form** - Validated form with Resend email integration
- **Responsive header** - Sticky navigation with branding and CTA button
- **Footer** - Social media links and copyright information

## Technology stack

<Tabs>
  <Tab title="Frontend">
    - **Next.js 15.5.9** - React framework with App Router
    - **React 19.2.1** - UI library
    - **TypeScript 5** - Type-safe development
    - **Tailwind CSS 3.4** - Utility-first CSS framework
    - **shadcn/ui** - Radix UI component library
  </Tab>
  <Tab title="Backend">
    - **Resend 3.5.0** - Email API for contact form
    - **Zod 3.24** - Schema validation
    - **React Hook Form 7.54** - Form state management
  </Tab>
  <Tab title="Deployment">
    - **Firebase App Hosting** - Google Cloud deployment
    - **Netlify** - Alternative deployment platform
    - **Turbopack** - Fast development builds
  </Tab>
</Tabs>
